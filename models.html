Reusable Research Resources

Open-Source Model Library

Models and algorithms for human spatiotemporal behavior, infectious-disease transmission, and intelligent intervention optimization. Each entry includes a concise overview, a representative figure, a related paper, and the source-code repository.

Synthetic Population & Trajectory Models

Methods for constructing individual travel patterns and population-level spatiotemporal distributions as inputs for urban simulation and epidemic modeling.

Human Mobility

M-STAR

M-STAR models human mobility through multi-scale spatiotemporal dependencies, supporting the characterization of intra-urban activity patterns and the generation of mobility scenarios.

Related paperM-STAR: Multi-Scale Spatiotemporal Autoregression for Human Mobility Modeling
GitHub Repository
Trajectory Generation

Traveller

Traveller uses an autoregressive diffusion framework to learn individual travel patterns and generate behaviorally plausible trajectories for simulation and intervention testing.

Related paperTraveller: Travel-Pattern Aware Trajectory Generation via Autoregressive Diffusion Models
GitHub Repository

Epidemic Transmission Models

Regional and individual-level approaches for representing population movement, contact structures, transmission dynamics, and intervention responses.

Regional Simulation

Metapopulation Model

The model divides a city into interconnected regions and combines interregional population movement with intraregional transmission to analyze spatial risk and region-level interventions.

Related paperImpact of Initial Outbreak Locations on Transmission Risk of Infectious Diseases in an Intra-Urban Area
GitHub Repository
Individual Simulation

Agent-Based Model

The individual-level model represents population attributes, spatial locations, contact structures, infection-state transitions, and behavioral responses to interventions.

Related paperA Data Driven Agent-Based Model that Recommends Non-Pharmaceutical Interventions to Suppress COVID-19 Resurgence in Megacities
GitHub Repository

Spatiotemporal Epidemic-Control Optimization

Reinforcement-learning and hierarchical decision models that balance transmission suppression, intervention costs, and decision robustness across spatial and temporal scales.

Deep Reinforcement Learning

DRL_EPC_STO

DRL_EPC_STO coordinates intervention intensity across regions and time periods while enforcing spatiotemporal orderliness, improving the efficiency of epidemic-control policies.

Related paperDeep-Reinforcement-Learning-Based Optimization for Intra-Urban Epidemic Control Considering Spatiotemporal Orderliness
GitHub Repository
Hierarchical Control

Hi_RICE_EPC

Hi_RICE_EPC combines region-level and individual-level decisions to optimize large-scale mobility interventions while balancing epidemic suppression against intervention costs.

Related paperArchitecting Urban Epidemic Defense: A Hierarchical Region-Individual Control Framework
GitHub Repository
Incomplete Information

EpidemicControl-IncompleteObs

This model addresses intervention optimization when the true epidemic state cannot be fully observed, producing robust policies under incomplete observations and uncertain information.

Related paper Epidemic Control under Incomplete Observations through Mechanistic State Reconstruction and Multi-Agent Learning (under review)
GitHub Repository
